Scenario
  • ‘Court to community’ sentencing
  • The service user is previously known to probation
  • Low-risk case (e.g. traffic or public order offence), managed by the National Probation Service that’s already been allocated to a probation practitioner in an office in Wales
  • No pre-sentence report (or oral/short sentence pre-sentence report)
  • No rehabilitation, just unpaid work (using existing mechanisms)
  • Using the existing mechanisms and processes for risk assessment
  • Main user need: Probation professionals need to record events to document the service user’s progress against a sentence
  • Main user need: Probation professionals need to schedule contacts with the service user
  • A way to hand the case back to the as-is process if the risk profile changes e.g there is a breach
  • A slice that is deliverable into (some) users’ hands by 1 Jan 2021.
